### Table to Cypress Data Attacks

Get the HTML table from the app, full HTML of the table to convert.
You'll have to do this a few times since the damages are broken into separate tables.

Usage Instructions:
Method 1: Node.js (Recommended)
Install jsdom (if you don't have it):
jsdom
Edit the script:
Open html-table-parser.js
Replace the content of the htmlTable variable with your HTML table
Save the file
Run the script:
js
Copy the output and paste it into your test file
Method 2: Browser Console
Open your browser's developer console
Copy and paste the entire script into the console
Replace the htmlTable variable with your HTML table
Press Enter to run it
